# Cybersecurity Log Analysis Tool

This project demonstrates basic security log analysis using Python.

## Project Purpose
The script analyzes authentication logs and detects possible brute-force login attempts.

## Files
auth_log.txt – Sample authentication log file  
log_detector.py – Python script that analyzes failed login attempts  
test.py – Testing script

## Detection Logic
The script counts failed login attempts in the log file.

If failed attempts ≥ 3, the script generates an alert:

⚠ Possible brute-force attack detected
